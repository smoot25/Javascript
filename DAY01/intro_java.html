   콘솔 출력과 자료형   

window.console    or    console

개발자 도구가 아닌 제작 환경에서 자바스크립트를 사용할 수 있게 만들어 줌.


자바에서 데이터표현시, 사용되는 형식을 자료형이라고 한다.

  • 숫자
  • 문자열
  • 불리언
  • 다양한 목적과 형태의 객체

  • console.log()


   변수와 상수   

[ let 변수이름; ] : 변수선언

[ 변수이름 = 데이터; ] : 변수 초기화

[ const 상수이름 = 데이터; ] : 변하지 않는값으로 지정.

변수는 변하는 값 상수는 변하지 않는 것.

상수에 변수 데이터 입력시 에러 발생함. ( 상수는 변하지 않는 값이기 때문 )


   문자열 prompt   

let string1 = '문자열에는 기호가 들어간다'

let string = '12345...?'

문자열을 더하여 더 긴 문자열로 바꾸는 것도 가능

[ prompt("문자열"); ] = 다이얼로그 박스에 띄울 메세지를 문자열로 등록가능.

const data = prompt() >>> 사용자 입력 >>> const data = "사용자 입력 내용"


   템플릿 리터럴   

const str3 = `백틱`

변수를 문자열으로 삽입하여 문자열 취급도 가능 : ${변수}

prompt 반환값도 ${prompt=''}를 이용하여 문자열 취급 가능.

` 기호를 사용하지 않으며 ${변수} 기능도 적요되지 않음.


   unfined 외   

null : 데이터가 없다는 것을 선언

undefind : 아직 값이 정의되지 않은 상태

boolean : True or False

confirm : 사용자가 확인과 취소중 하나를 선택. 선택에 따라 true or false


   DOM   
  • 렌더링의 순서
    1. HTML을 로드
    2. 그 내부 구조에 DOM을 형성
    3. 형성한 DOM을 기반으로 화면을 제작
  • DOM : 문서 객체 모델
  • DOM은 웹 콘텐츠 추가, 수정, 삭제 와 마우스, 키보드를 통한 이벤트 처리를 담당함.
  • document : DOM에 대한 접근을 할 수 있게하는 진입접이 됨.
  • querySelector : 선택자를 인자로 전달받고 일치하는 문서내의 첫번째 요소 반환
  • textContnet : 해당 요소가 포함한 내용을 확인하고 수정 가능하다.
